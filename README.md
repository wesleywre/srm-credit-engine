# SRM Credit Engine

Plataforma de cessão de crédito multimoedas (BRL/USD): precificação de recebíveis com deságio baseado em risco e liquidação transacional auditável.

> Case técnico — SRM Asset · Engenheiro de Software Backend Sênior.
> **Em desenvolvimento** — acompanhe a evolução pelas Pull Requests.

## Stack

- **Backend:** Python 3.12 · FastAPI · SQLAlchemy 2 (async) · PostgreSQL 16
- **Frontend:** React 19 · Vite · TypeScript · Tailwind CSS
- **Infra:** Docker Compose · GitHub Actions
